Skin Color Editor

The skin color editor is used for substituting and excluding colors in the current skin.

Color Selection

Click in the color selection palette to select the current color. This is also the color that will be used by the painting tools. If you double click, then the selected color in the substitute selection palette on the right will be put on the same color as the selected color in the color selection palette.

Substitute Selection

Select a color in the right color palette to use as a substitute for the color selected in the left color palette. The color selected on in the left palette is substituted by the color in the right palette after the substitute colors button is pressed. Double clicking in the substitution selection palette will select the same color in the left palette as is selected in the substitution color selection palette .

Substitute Colors

This button will be enabled when the selected color in the left color palette is being used in the currently selected skin. Then you select the color in the right side palette to substitute it with. Now pressing the "<<" button will perform the color substitution.

Color Excluding

Color excluding lets you exclude ranges of colors from the current skin.

You specify which color ranges to exclude by un-checking the check box for a specific color range. After you have specified what ranges need to be excluded, pressing the re-map button will re-map all excluded colors to included colors, that best match the specific excluded color.

Re-map Colors

This button will re-map the excluded colors in the skin to included colors, that best match the specific excluded color. To specify what colors to include and exclude, set the color exclusion check boxes.
